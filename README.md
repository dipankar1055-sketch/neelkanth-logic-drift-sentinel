## The #MehendiPrinciple

Our architecture is built on the **#MehendiPrinciple**: the belief that in high-stakes financial operations, human accountability must be indelible. Every algorithmic decision made by the Sentinel is linked to a human-attested verification, creating an unbreakable chain of custody for every trade and billing case.

## Architecture & Resilience

* **Sentinel Core**: Monitors transaction streams for logic deviations.
* **Delta Audit Logs**: Real-time logging of all drift events and human overrides.
